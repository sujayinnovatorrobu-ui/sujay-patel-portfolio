# Sujay Patel — Robotics & Automation Portfolio

I am a Robotics & Automation student at Government Polytechnic, Ahmedabad. This is my developer portfolio themed as an interactive engineering drawing sheet.

Live Site: https://sujayr07.netlify.app/
GitHub: https://github.com/sujayinnovatorrobu-ui/sujay-patel-portfolio

---

## Technical Projects & Deployments

- **Vayu Rakshak (Air Steriliser):** Built during the 2021 COVID crisis, integrating UVC chambers and HEPA H13 filters. Deployed in public clinics.
- **Earlobe Hemoglobin Analyser:** ESP32 optoelectronic screening device using multi-wavelength photoplethysmography (PPG).
- **Autonomous Warehouse Rover:** Differential-drive logistics robot with LiDAR and custom BLDC motor hubs. Runs ROS.
- **Maze-Solving Robot:** Autonomous maze-solving robot utilizing flood-fill algorithms (₹10,00,000 Robofest Gujarat 3.0 Champion).

*All championship winnings were directly reinvested to fund our team's startup, **Project & Innovation Lab**, in Ahmedabad.*

## Design Inspirations & Sources
This project draws functional and layout inspiration from:
- **Sheet layout & technical boundaries:** Adapted from the [Codrops Blueprint Layouts](https://tympanus.net/codrops/) design pattern.
- **Custom Cursor & Ribbons:** The coordinate-line crosshair tracker and hazard tapes are inspired by [Design Kettle](https://designkettle.in/).
- **Telemetry System:** Powered by the open-source [Hack Club Hackatime](https://github.com/hackclub/hackatime) API.
